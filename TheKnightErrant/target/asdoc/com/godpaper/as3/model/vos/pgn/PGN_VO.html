Classpublic class PGN_VO
InheritancePGN_VO Object

PGN(Portable Game Notation) VO.是棋类游戏过程的文件格式，既然国际象棋以这个规范作为记录棋谱的标准，那么对于中国象棋来说，在还没有一个统一标准的今天，PGN无疑是一个好的选择。

See also

http://www.xqbase.com/protocol/cchess_pgn.htm
